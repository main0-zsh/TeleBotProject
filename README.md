# TeleBotProject 🤖

Коллекция из 6 Telegram-ботов, написанных на Python. Проект демонстрирует развитие навыков от базовых команд до интеграции с внешними API и искусственным интеллектом.

## 📁 Состав проекта

### 1. Currency Converter (v1)
Базовый бот для конвертации валют. Позволяет быстро узнать курс и пересчитать сумму.

### 2. Advanced Currency Converter (v2)
Улучшенная версия конвертера с более удобным интерфейсом, поддержкой большего количества валют и актуальными данными из внешних источников.

### 3. Weather Bot ☁️
Бот, который сообщает текущую погоду в любом городе. Использует API OpenWeatherMap (или аналог) для получения данных в реальном времени.

### 4. Gemini AI Bot ✨
Интеграция с нейросетью **Google Gemini**. Бот способен поддерживать диалог, отвечать на сложные вопросы и помогать с написанием кода прямо в Telegram.

### 5. Water Tracker 💧
Полезный бот-помощник для здоровья. Помогает пользователю отслеживать дневную норму потребления воды и присылает напоминания.

### 6. Command & UI Bot 🔘
Демонстрация работы с интерфейсом Telegram: использование обычных (`Reply`) и встроенных (`Inline`) кнопок, меню и различных команд для взаимодействия с пользователем.

## 🛠 Технологии
* **Python 3**
* **aiogram** (или `pyTelegramBotAPI`)
* **Requests / Aiohttp** (для работы с API)
* **API Интеграции:** Gemini AI, Weather API, Exchange Rates API.

## 🚀 Как запустить

1. **Клонируйте репозиторий:**
   ```bash
   git clone [https://github.com/ваш-логин/TeleBotProject.git](https://github.com/ваш-логин/TeleBotProject.git)
   cd TeleBotProject
Настройте токены:
Создайте файл tokens.py в корне проекта и добавьте туда свои ключи (этот файл игнорируется гитом для безопасности):

Python
BOT_TOKEN = "ваш_токен_бота"
GEMINI_KEY = "ваш_ключ_gemini"
WEATHER_API_KEY = "ваш_ключ_погоды"
Установите зависимости:

Bash
./.venv/bin/python3 -m pip install aiogram requests python-dotenv
Запуск:
Запустите нужного бота, например:

Bash
./.venv/bin/python3 bot_gemini.py
